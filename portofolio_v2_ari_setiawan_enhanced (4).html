Business Automation · Python
Otomatisasi Analytics Business
Python · SQLAlchemy · PyMySQL · Pandas · smtplib · Cron Job
Problem
Proses pemantauan performa bisnis bersifat reaktif dan manual. Setiap minggu/bulan, staf harus menjalankan kueri ke database, mengunduh hasilnya, melakukan pembersihan data di Excel, dan menyusun email secara manual untuk manajer — menyebabkan risiko keterlambatan laporan dan potensi human error dalam perhitungan metrik kunci yang bisa menyesatkan pengambilan keputusan.
Solution
Mengembangkan skrip Python yang terhubung langsung ke database via SQLAlchemy / PyMySQL untuk ekstraksi data otomatis. Menggunakan Pandas untuk agregasi metrik, pembersihan data, dan analisis tren. Mengintegrasikan smtplib & email.mime untuk format laporan, serta Cron Job / Windows Task Scheduler agar laporan terkirim ke Gmail manajer secara terjadwal tanpa intervensi manusia.
Result
Mengubah proses pelaporan dari beberapa jam menjadi 0 menit waktu manual. Laporan tiba di kotak masuk manajer tepat waktu secara konsisten (misal: setiap Senin jam 08.00 pagi), meningkatkan kecepatan respons terhadap perubahan performa bisnis secara signifikan.

Sistem Prediksi Stok Otomatis
Python · Random Forest · Linear Regression · PDF Automation
Problem
Perusahaan mengalami kelebihan atau kekurangan stok secara berulang karena proses perencanaan inventori dilakukan secara manual dan intuitif tanpa basis data historis.
Solution
Membangun sistem prediksi kebutuhan stok 30 hari menggunakan algoritma Random Forest & Linear Regression, dilengkapi laporan PDF yang ter-generate otomatis setiap siklus.
Result
Akurasi perencanaan stok meningkat signifikan — tim supply chain dapat mengantisipasi kebutuhan sebulan ke depan dengan laporan yang siap distribusi tanpa input manual.

Masalah
Amazon India mengalami penurunan revenue 18.45% dalam 3 bulan. Tidak jelas apakah disebabkan harga, volume, kualitas produk, atau channel distribusi — dataset berisi 21 kolom dengan 2 kolom kosong sepenuhnya dan 110.489 baris data transaksi.
Proses
Audit menyeluruh 21 kolom, identifikasi missing values, analisis tren MoM growth, screening semua kategori produk untuk menemukan yang tumbuh di tengah penurunan, dan penyusunan Business Insight Report 6 bagian profesional.
Hasil
Membuktikan penurunan murni dari berkurangnya volume order (−22.7%), bukan harga (avg order value justru naik +5.6%). Menemukan Blazzer sebagai satu-satunya kategori tumbuh organik (+32.4%) → 5 rekomendasi strategis prioritas tinggi dengan estimasi dampak terukur.
